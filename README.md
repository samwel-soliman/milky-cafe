# Milky Café ☕

**Full-Stack Café Website**

Associated with **Arab Academy for Science, Technology and Maritime Transport**  
Developed as part of the **Web Design course at NTI**.

## Project Overview

Milky Café is a full-stack café website designed to represent a French-style café offering pastries and beverages.

The project focuses on delivering a smooth, elegant, and user-friendly web experience while combining modern front-end design with backend functionality using PHP.

The website includes multiple pages, responsive layouts, reservation handling, contact forms, and basic database integration to simulate a real-world business website.

## Key Features

- Multi-page responsive website
- Home, Menu, Reservation, Contact, About, FAQ, and Vision pages
- Reservation system with form handling using PHP
- Interactive UI with smooth navigation and animations
- Menu categorization for bakery items and hot/cold drinks
- Customer feedback and contact form handling
- Intro video page with skip functionality
- Basic MySQL database integration

## Technologies Used

- HTML5
- CSS3
- JavaScript ES6
- jQuery
- Bootstrap
- PHP
- MySQL
